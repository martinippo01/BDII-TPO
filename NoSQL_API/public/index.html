API Documentation

Clients

GET /clients

Retrieve a list of clients.

Example Request:

GET /clients

Example Response:

[
    {
        "nro_cliente": 1,
        "nombre": "John",
        "apellido": "Doe",
        "direccion": "123 Main St",
        "activo": true
    },
    {
        "nro_cliente": 2,
        "nombre": "Jane",
        "apellido": "Smith",
        "direccion": "456 Elm St",
        "activo": true
    }
]

POST /clients

Create a new user.

Request Body (JSON):

{
    "nombre": "John",
    "apellido": "Doe",
    "direccion": "123 Main St",
    "activo": true
}

Example Request:

POST /clients
Content-Type: application/json

{
    "nombre": "John",
    "apellido": "Doe",
    "direccion": "123 Main St",
    "activo": true
}

Example Response:

{
    "nro_cliente": 3,
    "nombre": "John",
    "apellido": "Doe",
    "direccion": "123 Main St",
    "activo": true
}

PUT /clients/:id

Update an existing user by ID.

Request Body (JSON):

{
    "nombre": "Updated Name",
    "apellido": "Updated LastName",
    "direccion": "Updated Address",
    "activo": false
}

Example Request:

PUT /clients/3
Content-Type: application/json

{
    "nombre": "Updated Name",
    "apellido": "Updated LastName",
    "direccion": "Updated Address",
    "activo": false
}

Example Response:

{
    "nro_cliente": 3,
    "nombre": "Updated Name",
    "apellido": "Updated LastName",
    "direccion": "Updated Address",
    "activo": false
}

DELETE /clients/:id

Delete a user by ID.

Example Request:

DELETE /clients/3

Example Response:

{
    "message": "User deleted"
}

Products

GET /products

Retrieve a list of products.

Example Request:

GET /products

Example Response:

[
    {
        "nro_producto": 1,
        "marca": "Brand A",
        "nombre": "Product 1",
        "descripcion": "Description 1",
        "precio": 99.99,
        "stock": 100
    },
    {
        "nro_producto": 2,
        "marca": "Brand B",
        "nombre": "Product 2",
        "descripcion": "Description 2",
        "precio": 79.99,
        "stock": 150
    }
]

POST /products

Create a new product.

Request Body (JSON):

{
    "marca": "Brand A",
    "nombre": "Product 1",
    "descripcion": "Description 1",
    "precio": 99.99,
    "stock": 100
}

Example Request:

POST /products
Content-Type: application/json

{
    "marca": "Brand A",
    "nombre": "Product 1",
    "descripcion": "Description 1",
    "precio": 99.99,
    "stock": 100
}

Example Response:

{
    "nro_producto": 3,
    "marca": "Brand A",
    "nombre": "Product 1",
    "descripcion": "Description 1",
    "precio": 99.99,
    "stock": 100
}

GET /products/:id

Retrieve product details by ID.

Example Request:

    GET /products/1

Example Response (Product Found):

    {
        "nro_producto": 1,
        "marca": "Brand A",
        "nombre": "Product 1",
        "descripcion": "Description 1",
        "precio": 99.99,
        "stock": 100
    }

Example Response (Product Not Found):

    {
        "error": "Product not found"
    }
