API Documentation

Users

GET /users

Retrieve a list of users.

Example Request:

GET /users

Example Response:

[
    {
        "nro_cliente": 1,
        "nombre": "John",
        "apellido": "Doe",
        "direccion": "123 Main St",
        "activo": true
    },
    {
        "nro_cliente": 2,
        "nombre": "Jane",
        "apellido": "Smith",
        "direccion": "456 Elm St",
        "activo": true
    }
]

POST /users

Create a new user.

Request Body (JSON):

{
    "nombre": "John",
    "apellido": "Doe",
    "direccion": "123 Main St",
    "activo": true
}

Example Request:

POST /users
Content-Type: application/json

{
    "nombre": "John",
    "apellido": "Doe",
    "direccion": "123 Main St",
    "activo": true
}

Example Response:

{
    "nro_cliente": 3,
    "nombre": "John",
    "apellido": "Doe",
    "direccion": "123 Main St",
    "activo": true
}

PUT /users/:id

Update an existing user by ID.

Request Body (JSON):

{
    "nombre": "Updated Name",
    "apellido": "Updated LastName",
    "direccion": "Updated Address",
    "activo": false
}

Example Request:

PUT /users/3
Content-Type: application/json

{
    "nombre": "Updated Name",
    "apellido": "Updated LastName",
    "direccion": "Updated Address",
    "activo": false
}

Example Response:

{
    "nro_cliente": 3,
    "nombre": "Updated Name",
    "apellido": "Updated LastName",
    "direccion": "Updated Address",
    "activo": false
}

DELETE /users/:id

Delete a user by ID.

Example Request:

DELETE /users/3

Example Response:

{
    "message": "User deleted"
}

Products

GET /products

        "nro_producto": 1,
        "marca": "Brand A",
        "nombre": "Product 1",
        "descripcion": "Description 1",
        "precio": 99.99,
        "stock": 100
    },
    {
        "nro_producto": 2,
        "marca": "Brand B",
        "nombre": "Product 2",
        "descripcion": "Description 2",
        "precio": 79.99,
        "stock": 150
    }
]

Request Body (JSON):

{
